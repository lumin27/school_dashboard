A full-featured platform to help schools manage students, teachers, parents, classes, and attendance efficiently.

## Problem Solved

Schools often struggle to track attendance, communicate with parents, and manage multiple roles. EduTrack centralizes these operations in one easy-to-use platform.

## Features

- Role-based access: Admin, Teacher, Student, Parent
- Attendance marking (individual or mark all present/absent)
- Messaging between users based on role
- Dashboard with charts for attendance visualization
- CRUD operations for students, teachers, classes, and lessons

## Tech Stack

- **Frontend/Backend:** Next.js (App Router)
- **Database:** PostgreSQL with Prisma
- **Authentication:** Clerk
- **Image Storage:** Cloudinary
- **Deployment:** Vercel
